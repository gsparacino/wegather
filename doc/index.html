Il Progetto

Progetto sviluppato per il corso “Laboratorio di Progettazione” del secondo anno della Magistrale Informatica presso l’Università di Milano Bicocca.

Descrizione generale

L’applicazione ha lo scopo di fornire uno strumento intuitivo e semplice per l’organizzazione di eventi di gruppo permettendo a tutti gli invitati di proporre e votare sia le date che le attività che desiderano svolgere. Il software è stato sviluppato a partire da un progetto già esistente che non soddisfava tutte le funzionalità richieste ma ha fornito un buon punto di partenza.

Informazioni Tecniche

Repository: https://gitlab.com/g.sparacino3/wegather

Architettura: MVC

Linguaggio: Java

Framework: SpringBoot

Database: H2 in memory

Dipendenze

Hibernate

Tool utilizzato per l'Object-relational mapping (ORM) tra il database relazionale utilizzato (H2) e le classi Java. La scelta è ricaduta su questa tecnologia per l’elevata compatibilità e integrazione con il framework SpringBoot.

Thymeleaf

Java Template engine utilizzato per implementare il pattern "Template-View" permettendo di creare pagine web dinamiche a partire da template html. Scelto perchè molto ben integrato con Spring Framework.

H2

Databese SQL per lo storage dei dati dell’applicazione. La scelta è ricaduta su questa tecnologia poiché open Soruce, gratuita ma molto veloce per carichi di lavoro e quantità di dati richiesti dall’applicazione.

Jekill

Generatore di siti statici utilizzato per semplificare la costruzione della documentazione. La scelta è ricaduta su questa tecnologia perchè permette di definire un layout che viene applicato automaticamente a tutte le pagine del sito senza che se ne preoccupi chi scrive i contenuti, è inoltre capace di generare il sito sia a partire da file html che .md.

GitLab Pages

Il progetto sfrutta pages per la pubblicazione della documentazione sotto forma di sito statico. La scelta è ricaduta su questa tecnologia poichè facilmente integrabile nella pipeline di GitLab e permette quindi di mantenere la documentazione automaticamente aggiornata.
